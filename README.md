# Trader-Behavior-Insights

# Project Overview
This project explores the relationship between historical trading data from Hyperliquid and Bitcoin Market Sentiment (Fear & Greed Index). The objective was to identify behavioral patterns and performance metrics that can drive smarter, sentiment-aware trading strategies in Web3 markets.

# Technical Implementation
Environment: Python 3.11+

Libraries: Pandas, NumPy, Matplotlib, Seaborn.

Data Processing: * Standardized disparate time formats across datasets.

Filtered realized PnL trades to calculate accurate Win Rates.
